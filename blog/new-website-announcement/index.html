New Website - Announcement

April 18th, 2022

It's been a long time coming - I've updated my website again!


As you'll notice, all links have stayed the same, and nothing is much more different to navigate. So what's the deal, Mike? What's actually new?


Well, for starters, I've completely remade everything from scratch. Which means I'm no longer using Wix.


Was this a smart idea? Maybe. Not sure yet. But Wix is a lot more expensive than I'm willing to put up with for something that I can do myself with roughly the same quality.


Other than a few style differences, like the main font of the website, you'll also notice that all of the sections are completed, like the resume and portfolio. I've never worked on a website like this from scratch on such a massive scale, so if you notice something doesn't look quite right, find some way to message me!


Inevitably, this was just a cheaper way for me to be happy with how my website turns out and still have total control over every aspect. Things will probably keep changing, but there won't be any new announcement about it unless something big happens.
